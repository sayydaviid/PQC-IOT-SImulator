# PQC IoT Simulator

Simulador para avaliação de criptografia pós quântica em redes IoT.

A plataforma usa Mininet e Mininet WiFi para emulação da rede e NetworkX apenas para visualização da topologia.

## Dependencies

- Python dependencies: pqc_iot_sim/requirements-project.txt
- Mininet and Mininet-WiFi are system-level dependencies; install them separately before running the engine.
- The file pqc_iot_sim/requirements.txt is a snapshot of the local environment and should not be used to install project dependencies.
